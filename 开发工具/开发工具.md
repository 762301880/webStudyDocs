## VS Code 常用快捷键

### 一、基础文件操作

Ctrl+Shift+P 打开命令面板

Ctrl+N 新建文件

Ctrl+S 保存文件

Ctrl+Shift+S 另存为

Ctrl+W 关闭当前编辑页

Ctrl+Z 撤销

Ctrl+Y 重做

### 二、查找替换

Ctrl+F 当前文件内查找

Ctrl+H 当前文件替换

Ctrl+Shift+F 全局跨文件搜索

Ctrl+Shift+H 全局批量替换

F3 下一个匹配内容

Shift+F3 上一个匹配内容

### 三、代码行编辑（最高频）

Ctrl+/ 单行注释

Shift+Alt+A 块注释

Shift+Alt+↓ 向下复制一行

Shift+Alt+↑ 向上复制一行

Alt+↓ 整行下移

Alt+↑ 整行上移

Ctrl+Shift+K 删除整行

Shift+Alt+F 自动格式化代码

Tab 增加缩进

Shift+Tab 减少缩进

### 四、多光标批量编辑

Ctrl+D 选中下一个相同单词，生成多光标

Ctrl+K 再按 Ctrl+D 跳过当前匹配

Ctrl+Shift+L 选中文档所有相同文本

Alt + 鼠标点击 手动添加多个光标

Ctrl+L 选中整行代码

### 五、窗口侧边栏与终端

Ctrl+` 调出 / 隐藏终端

Ctrl+Shift+E 打开文件资源管理器

Ctrl+Shift+X 打开扩展商店

Ctrl+\ 拆分编辑器分屏

Ctrl+1 / Ctrl+2 / Ctrl+3 切换不同分屏

F11 全屏切换

Ctrl + 加号 放大界面字体

Ctrl + 减号 缩小界面字体



### 六、代码跳转导航

Ctrl+P 快速打开项目任意文件

Ctrl+G 跳转到指定行数

F12 跳转到函数 / 变量定义

Alt+F12 悬浮查看定义，不跳转

Alt + 左箭头 返回上一次编辑位置

Alt + 右箭头 前进到下一个位置

Ctrl+Shift+O 打开文件大纲，快速跳转函数

Ctrl+Tab 切换打开的标签页

### 七、代码折叠

Ctrl+Shift+[ 折叠当前代码块

Ctrl+Shift+] 展开当前代码块

Ctrl+K 松开再按 Ctrl+0 全部折叠代码

Ctrl+K 松开再按 Ctrl+J 全部展开代码

## vscode如何快速建立html骨架快捷键

输入 `!` 后直接按 **Enter** 回车，同样展开骨架，部分输入法环境下更顺手。
