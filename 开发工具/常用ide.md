# vscode

## ubuntu-install-vscode 

> 进入vscode[官网](https://code.visualstudio.com/)[下载](https://code.visualstudio.com/Download)vscode
>
> **注:**<font color='yellow'>如果下载很慢请开外网下载</font>

- 使用命令安装下载的deb包

```shell
sudo dpkg -i   下载的vscode.deb包
```
