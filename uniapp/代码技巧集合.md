### 条件编译

条件编译是用特殊的注释作为标记，在编译时根据这些特殊的注释，将注释里面的代码编译到不同平台。

### 使用方法

以 `#ifdef` 或 `#ifndef` 加 `%PLATFORM%` 开头，以 `#endif` 结尾。

- `#ifdef`：if defined 仅在某平台存在
- `#ifndef`：if not defined 除了某平台均存在
- `%PLATFORM%`：平台名称

### 全局多端判断封装方案

#### 工具函数（推荐，全局导入使用）

新建文件 `src/utils/platform.js`

```js
/**
 * 获取当前运行端标识
 * @returns {string} app / h5 / wechat / alipay / douyin / qq / kuaishou / baidu
 */
export function getPlatform() {
  // APP端
  // #ifdef APP-PLUS
  return 'app'
  // #endif

  // H5网页
  // #ifdef H5
  return 'h5'
  // #endif

  // 微信小程序
  // #ifdef MP-WEIXIN
  return 'wechat'
  // #endif

  // 支付宝小程序
  // #ifdef MP-ALIPAY
  return 'alipay'
  // #endif

  // 抖音/头条小程序
  // #ifdef MP-TOUTIAO
  return 'douyin'
  // #endif

  // QQ小程序
  // #ifdef MP-QQ
  return 'qq'
  // #endif

  // 快手小程序
  // #ifdef MP-KUAISHOU
  return 'kuaishou'
  // #endif

  // 百度小程序
  // #ifdef MP-BAIDU
  return 'baidu'
  // #endif
}

/**
 * 获取传给后端的source来源字段
 * @returns {string} APP / H5 / WECHAT / ALIPAY / DOUYIN
 */
export function getSource() {
  // #ifdef APP-PLUS
  return 'APP'
  // #endif
  // #ifdef H5
  return 'H5'
  // #endif
  // #ifdef MP-WEIXIN
  return 'WECHAT'
  // #endif
  // #ifdef MP-ALIPAY
  return 'ALIPAY'
  // #endif
  // #ifdef MP-TOUTIAO
  return 'DOUYIN'
  // #endif
  return 'OTHER'
}

/**
 * 平台判断快捷方法
 */
export const platform = {
  isApp() {
    // #ifdef APP-PLUS
    return true
    // #endif
    return false
  },
  isH5() {
    // #ifdef H5
    return true
    // #endif
    return false
  },
  isWechat() {
    // #ifdef MP-WEIXIN
    return true
    // #endif
    return false
  },
  isAlipay() {
    // #ifdef MP-ALIPAY
    return true
    // #endif
    return false
  },
  isDouyin() {
    // #ifdef MP-TOUTIAO
    return true
    // #endif
    return false
  },
  // 是否任意小程序
  isMp() {
    // #ifdef MP
    return true
    // #endif
    return false
  }
}
```

#### **全局挂载 main.js（页面无需每次 import）**

```js
import { getPlatform, getSource, platform } from '@/utils/platform'

// 挂载到全局
uni.$getPlatform = getPlatform
uni.$getSource = getSource
uni.$platform = platform
```

#### **页面 / 组件中使用示例**

```js
export default {
  onLoad() {
    // 获取端标识
    const p = uni.$getPlatform()
    console.log('当前端', p)

    // 登录接口使用source
    const source = uni.$getSource()
    this.login(source)

    // 快捷布尔判断
    if (uni.$platform.isApp()) {
      console.log('APP端逻辑')
    }
    if (uni.$platform.isWechat()) {
      console.log('微信小程序逻辑')
    }
    if (uni.$platform.isMp()) {
      console.log('所有小程序通用逻辑')
    }
  },
  methods: {
    login(source) {
      console.log('提交来源', source)
      // api.login({ source })
    }
  }
}
```

#### 补充说明

1. 编译期无多余代码

   条件编译打包时只会保留当前平台的 return，其他分支直接删除，无运行时判断损耗；

2. 如果你想用在模板 template

   可以在页面 `computed`中映射全局方法：

```js
computed: {
  isApp() {
    return uni.$platform.isApp()
  }
}
```

```vue
<view v-if="isApp">仅APP展示</view>
```

运行时兼容兜底（可选扩展）

如果需要动态判断（不推荐，仅特殊场景），可以搭配 `uni.getSystemInfoSync` 做双重校验。

### vue3利用生命周期onLoad渲染数据示例

```vue
<template>
  <view>
    <view v-for="item in tableData" :key="item.id">
      {{ item.title }}
    </view>
  </view>
</template>

<script setup>
import { ref, onLoad } from 'vue'
// 导入uni页面生命周期
// import { onLoad } from '@dcloudio/uni-app'

// 定义列表数据
const tableData = ref([])

// onLoad 页面加载钩子
onLoad((option) => {
  console.log('跳转参数：', option)
  loadData()
})

// 获取接口数据
const loadData = async () => {
  // 模拟请求
  const apiRes = await uni.request({
    url: 'https://xxx.com/api/list'
  })
  tableData.value = apiRes.data
}
</script>
```
