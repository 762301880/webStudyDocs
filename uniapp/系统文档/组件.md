## 扩展组件

### [uni-icons 图标](https://uniapp.dcloud.net.cn/component/uniui/uni-icons.html)

**uni-icons使用阿里图标库[iconfont](https://www.iconfont.cn/)**

#### uniApp 引入阿里云图标库（iconfont）完整教程

一、准备阿里云图标

1. 打开 [阿里图标库 iconfont.cn](https://link.wtturl.cn/?target=https%3A%2F%2Fwww.iconfont.cn%2F&scene=im&aid=497858&lang=zh)，登录账号

2. 搜索需要图标，加入**购物车**，添加到**项目**

3. 进入项目，选择 **Font class** 模式（推荐，兼容性最好）

4. 点击「

   下载至本地

   」，解压得到文件：

   - iconfont.css
   - iconfont.ttf
   - iconfont.woff
   - iconfont.woff2
   - iconfont.svg

二、文件放入 uniApp 项目

在项目根目录新建文件夹 `static/iconfont`，把解压全部文件丢进去。

三、修改 iconfont.css 字体路径（关键！不然小程序 / H5 不显示）

打开 `static/iconfont/iconfont.css`，找到所有 `@font-face` 内 `src` 路径：

原代码（本地下载的相对路径，uniApp 识别失效）

```css
src: url('iconfont.ttf?t=xxxx') format('truetype'),
url('iconfont.woff?t=xxxx') format('woff'),
url('iconfont.woff2?t=xxxx') format('woff2'),
url('iconfont.svg?t=xxxx#iconfont') format('svg');
```

修改为 uniApp static 绝对路径

```css
src: url('/static/iconfont/iconfont.ttf?t=xxxx') format('truetype'),
url('/static/iconfont/iconfont.woff?t=xxxx') format('woff'),
url('/static/iconfont/iconfont.woff2?t=xxxx') format('woff2'),
url('/static/iconfont/iconfont.svg?t=xxxx#iconfont') format('svg');
```

全局引入样式

App.vue 全局样式引入

```vue
<style>
/* 引入图标样式 */
@import url("/static/iconfont/iconfont.css");
</style>
```

五、页面使用图标

> 两种方式都可以

```vue
	<uni-icons custom-prefix="iconfont" type="icon-youhuiquan" size="30"></uni-icons>
    <!-- iconfont 基础类名 + 图标class名 -->
	<text class="iconfont icon-youhuiquan" style="font-size:30px;color:#b71423;"></text>
```
