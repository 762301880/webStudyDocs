## 新建页面严厉禁止使用中文

### 错误写法（中文，不行）

```json
{
	"path": "demo/页面跳转",
	"style": {
		"navigationBarTitleText": ""
	}
}
```

### 正确写法（英文 / 拼音，必须）

```json
{
	"path": "demo/page-jump",
	"style": {
		"navigationBarTitleText": "页面跳转"
	}
}
```

### **路径 + 文件名不能用中文**，这是小程序 /uni-app 的硬性规定

标题 `navigationBarTitleText` 可以写中文

改完**重启运行**，页面马上能访问
