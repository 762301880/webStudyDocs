### onMounted() 示例

> 注册一个回调函数，在组件挂载完成后执行。
>
> 加载 = 组件初始化、创建实例（DOM 不存在）
>
> 挂载 = 页面真实 DOM 渲染完成、可见（onMounted）
>
> **可用来初始化渲染列表数据等**

```vue
<script setup>
import { ref, onMounted } from 'vue'
const box = ref(null)

// 1. 组件刚创建就执行，此时DOM还没渲染
console.log(box.value) // null

onMounted(() => {
  // 2. DOM已经渲染到页面，挂载完成后执行
  console.log(box.value) // 真实dom元素
})
</script>
<template>
  <div ref="box"></div>
</template>
```
