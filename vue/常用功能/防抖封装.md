## 防抖原理

防抖：**多次触发只执行最后一次**（搜索输入、窗口缩放、滚动、按钮重复点击）

节流：多次触发按固定间隔执行（下拉加载、拖拽）

## 通用防抖工具函数（新建 utils/debounce.js）

```js
/**
 * 防抖函数
 * @param {Function} fn 执行函数
 * @param {Number} delay 延迟毫秒
 * @param {Boolean} immediate 是否立即执行（true：第一次触发立刻执行，延迟内不再触发）
 */
export function debounce(fn, delay = 300, immediate = false) {
  let timer = null;
  return function (...args) {
    if (timer) clearTimeout(timer);
    // 立即执行模式
    if (immediate && !timer) {
      fn.apply(this, args);
    }
    timer = setTimeout(() => {
      timer = null;
      if (!immediate) fn.apply(this, args);
    }, delay);
  };
}
```

## 使用

### 1. 输入框搜索防抖

```vue
<template>
  <input v-model="keyword" @input="handleSearch" placeholder="搜索" />
</template>

<script setup>
import { ref } from 'vue'
import { debounce } from '@/utils/debounce'

const keyword = ref('')

// 防抖包装函数，300ms延迟
const handleSearch = debounce(async () => {
  console.log('请求接口', keyword.value)
  // await api.search(keyword.value)
}, 300)
</script>
```

### 2. 按钮防止重复点击（立即执行防抖）

```vue
<template>
  <button @click="submit">提交</button>
</template>

<script setup>
import { debounce } from '@/utils/debounce'

// immediate=true 首次立刻执行，短时间重复点击失效
const submit = debounce(() => {
  console.log('提交表单')
}, 1500, true)
</script>
```

### vue2使用

#### js

```vue
<template>
  <input v-model="text" @input="search" />
</template>

<script>
import { debounce } from '@/utils/debounce'
export default {
  data() {
    return { text: '' }
  },
  methods: {
    // 防抖包装
    search: debounce(function () {
      console.log(this.text)
    }, 300)
  }
}
</script>
```

## 坑点注意（高频踩坑）

### ❌ 错误写法（每次渲染生成新防抖，失效）

```js
// 绝对不要写在模板内联
<input @input="debounce(()=>{},300)" />
```
