## Vue 记录页面 / 视频阅读时长完整方案

分两种场景：**页面浏览时长（文章 / 页面停留）**、**视频播放观看时长**

原理：

1. 进入页面记录开始时间
2. 离开 / 切页 / 关闭标签 / 失焦时计算总时长
3. 防抖 + 定时器实时累加，防止后台休眠、页面隐藏计时不准
4. 离开前上报后端

### 一、通用页面阅读时长（Vue3 + Composition API）

```vue
<template>
  <div class="article-page">
    <h1>文章内容</h1>
    <p>正文内容......</p>
  </div>
</template>

<script setup>
import { onMounted, onUnmounted } from 'vue'
import { useRoute, onBeforeRouteLeave } from 'vue-router'

const route = useRoute()
// 配置
const REPORT_INTERVAL = 30000 // 每30s上报一次时长
let startTime = 0 // 进入页面时间戳
let totalWatchTime = 0 // 累计有效观看时长(ms)
let timer = null // 计时定时器
let reportTimer = null // 定时上报定时器
let isPageVisible = true // 页面是否可见

// 上报时长接口
const reportWatchTime = async (duration) => {
  if (duration <= 0) return
  console.log('上报观看时长(秒)', Math.floor(duration / 1000))
  // await axios.post('/api/user/watch-time', {
  //   articleId: route.params.id,
  //   watchSecond: Math.floor(duration / 1000)
  // })
}

// 计算当前有效时长
const calcDuration = () => {
  if (!startTime || !isPageVisible) return
  const now = Date.now()
  const diff = now - startTime
  totalWatchTime += diff
  startTime = now
}

// 开始计时
const startTimer = () => {
  startTime = Date.now()
  // 每1s累加时长
  timer = setInterval(() => {
    if (document.hidden) {
      isPageVisible = false
      calcDuration()
      return
    }
    isPageVisible = true
    calcDuration()
  }, 1000)

  // 定时批量上报
  reportTimer = setInterval(() => {
    reportWatchTime(totalWatchTime)
    totalWatchTime = 0 // 上报后清空累计
  }, REPORT_INTERVAL)
}

// 停止计时并立即上报
const stopTimer = () => {
  calcDuration()
  clearInterval(timer)
  clearInterval(reportTimer)
  timer = null
  reportTimer = null
  reportWatchTime(totalWatchTime)
}

// 监听浏览器标签显示/隐藏
const visibilityChange = () => {
  if (document.hidden) {
    // 切后台/最小化，暂停计时
    calcDuration()
    isPageVisible = false
  } else {
    // 切回页面，重新记录起点
    startTime = Date.now()
    isPageVisible = true
  }
}

onMounted(() => {
  startTimer()
  document.addEventListener('visibilitychange', visibilityChange)
})

// 路由跳转离开页面
onBeforeRouteLeave(() => {
  stopTimer()
  document.removeEventListener('visibilitychange', visibilityChange)
})

// 组件销毁
onUnmounted(() => {
  stopTimer()
  document.removeEventListener('visibilitychange', visibilityChange)
})
</script>
```

### 二、视频播放精准观看时长（带播放 / 暂停 / 拖动过滤）

只统计**有效播放时长**，快进、暂停、切后台不计入观看时长

```vue
<template>
  <video
    ref="videoRef"
    src="/xxx.mp4"
    @play="handlePlay"
    @pause="handlePause"
    @timeupdate="handleTimeUpdate"
    @seeked="handleSeek"
  ></video>
</template>

<script setup>
import { ref, onMounted, onUnmounted } from 'vue'
const videoRef = ref(null)

let lastPlayTime = 0 // 上一帧播放时间点
let totalVideoWatch = 0 // 有效观看秒数
let isPlaying = false

// 上报视频时长
const reportVideoTime = (second) => {
  console.log('视频有效观看时长', second)
  // axios.post('/api/video/log', { videoId: 1, watchSecond: second })
}

// 播放
const handlePlay = () => {
  isPlaying = true
  lastPlayTime = videoRef.value.currentTime
}

// 暂停
const handlePause = () => {
  isPlaying = false
  // 暂停时结算时长
  const diff = videoRef.value.currentTime - lastPlayTime
  if (diff > 0 && diff < 30) { // 过滤拖动跳变过大
    totalVideoWatch += diff
  }
  reportVideoTime(Math.floor(totalVideoWatch))
}

// 实时播放进度更新
const handleTimeUpdate = () => {
  if (!isPlaying) return
  const current = videoRef.value.currentTime
  const diff = current - lastPlayTime
  // 正常播放diff≈1，拖动会突变，过滤异常值
  if (diff > 0 && diff < 3) {
    totalVideoWatch += diff
  }
  lastPlayTime = current
}

// 用户拖动进度条
const handleSeek = () => {
  lastPlayTime = videoRef.value.currentTime
}

onUnmounted(() => {
  // 退出页面最终上报
  reportVideoTime(Math.floor(totalVideoWatch))
})
</script>
```

### 后端存储建议

数据表设计示例：

```sql
CREATE TABLE user_watch_log (
  id INT PRIMARY KEY AUTO_INCREMENT,
  user_id INT COMMENT '用户ID',
  target_id INT COMMENT '文章/视频ID',
  type TINYINT COMMENT '1文章 2视频',
  watch_second INT COMMENT '观看秒数',
  create_time DATETIME
);
```
